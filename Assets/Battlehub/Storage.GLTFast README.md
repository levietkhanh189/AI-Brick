# GTLFast external asset loader for Runtime Asset Database

1. Install the glTFast (5.0.4) package as described [here](https://github.com/atteneder/glTFast#installing)

2. Import "Storage.GLTFast" package

3. Open the "ImportExternalGLBExample" scene located in  Assets\Battlehub.Extensions\Storage.GLTFast.Samples\ImportExternalGLBExample

4. Select "ImportExternalGLBExample" and change the path to point to your GLB file

5. Click Play
